## 🚀 Começando

### 📋 Pré-requisitos

  É necessário ter npm e node instalados no computador

### 🔧 Instalação

_Para instalar este projeto você precisa:_

1. Clonar o repositório
   ```sh
   git clone https://guiqsassi/Calculator.git
   ```
2. Instalar os pacotes
   ```sh
   npm i 
   ```
